## <a name="bugs"></a>Bugs resolved

* **_Tests_: Fix compatibility with newest upgrade-tests ([#35123](https://github.com/civicrm/civicrm-core/pull/35123))**
* **_Composer_: Set version of "brick/money" more precisely ([dev/core#6362](https://lab.civicrm.org/dev/core/-/issues/6362))**

  The package "brick/money" had a recent release that will fail to work with current CiviCRM. For composer-based sites (Drupal 9+), this impacts install/upgrade.
